# MLFlow Project

An MLOps/DevOps project for machine learning workflow automation, model management, and CI/CD integration.

## Project Structure

```
mlops-devops-project/
│
├── app/
│   └── app.py
│
├── model/
│
├── data/
├── create_dataset.py
├── register_model.py
├── requirements.txt
├── Dockerfile
├── .gitignore
├── README.md
└── dvc.yaml
```

## Getting Started

1. Install dependencies: `pip install -r requirements.txt`
2. Create datasets: `python create_dataset.py`
3. Register models: `python register_model.py`
4. Run the application: `python app/app.py`
